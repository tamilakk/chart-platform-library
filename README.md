# Chart Platform Library

Multiplatform open-source library for generating charts in React on the client side and as static images on the backend.

## Planned structure

- `packages/core` – shared types and normalization
- `packages/react-renderer` – React chart rendering
- `packages/server-renderer` – server-side SVG/PNG export
- `apps/demo` – demonstration app
